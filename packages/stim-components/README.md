# @stim-io/stim-components

Shared atomic Vue components, tokens, and theme CSS for the `stim` product surface.

## Install

This package is published through GitHub Packages.

Configure your npm registry for the `@stim-io` scope, then install:

```bash
pnpm add @stim-io/stim-components
```

## Exports

- package entrypoint: `@stim-io/stim-components`
- base styles: `@stim-io/stim-components/style.css`
- tokens: `@stim-io/stim-components/tokens.css`
- theme CSS:
  - `@stim-io/stim-components/themes/light/common.css`
  - `@stim-io/stim-components/themes/light/chromium.css`
  - `@stim-io/stim-components/themes/light/webkit.css`
  - `@stim-io/stim-components/themes/dark/common.css`
  - `@stim-io/stim-components/themes/dark/chromium.css`
  - `@stim-io/stim-components/themes/dark/webkit.css`

## Scope

This package owns atomic component and theme primitives.

Product-specific screen composition stays in `stim`.
